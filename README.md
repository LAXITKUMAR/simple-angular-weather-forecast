# simple-angular-weather-forecast
Simple Angular Weather Forecast

Steps to build and run the project in dev environment:
    ``` npm start ```

run the project in prod environment:
    ``` gulp serve:prod ```

Steps to run test:
    ``` npm test ```

Steps to run test(single run):
    ``` npm run test-single-run ```

Steps to run E2E tests:
    ``` npm run test:e2e ```
